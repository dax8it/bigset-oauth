# BigSet Hermes Launch Video — Visual Identity

## Style Prompt

Dark technical launch film for X: precise, high-contrast, operator-grade. It should feel like a working local agent stack coming online, not a corporate SaaS explainer.

## Mood

- Local-first infrastructure
- Agentic research system
- Codex OAuth power without key sprawl
- Verified data pipeline, not demo vapor
- Agent-led BigSet skill workflow: discovery questions before dataset creation, expected prompt structure, no prompt drift
- Client-ready delivery: BigSet link, CSV/XLSX, HTML/PDF report, and optional SMTP email to yourself or a client

## Palette

- Canvas black: `#05070B`
- Deep navy: `#08111F`
- Hermes cyan: `#5EEAD4`
- Codex violet: `#A78BFA`
- BigSet amber: `#FBBF24`
- Text white: `#F8FAFC`
- Muted slate: `#94A3B8`
- Success green: `#22C55E`

## Typography

- Primary: `Inter`, `SF Pro Display`, `system-ui`, sans-serif
- Mono: `JetBrains Mono`, `SF Mono`, `Menlo`, monospace

## Motion Rules

- Fast command-line reveals, gentle graph/data-flow motion.
- Cards slide and scale into alignment like infrastructure blocks snapping into place.
- No random motion; deterministic looping background only.
- Emphasize the architecture: BigSet, Hermes Agent, Codex OAuth, web tools, Convex.

## What NOT to Do

- No generic blue SaaS gradients.
- No fake screenshots or fabricated terminal output.
- No cluttered wall of text.
- No PowerPoint title-card slideshow.
- No claims that Hermes receives database write tools.
